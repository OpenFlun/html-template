## [4.0.0] - 2026-04-26 22:00
### 重大更新:
1. 启动服务器参数改为对象形式,并且新增是否启用登录:
```js
startDevServer({ port: 7296, hotReload: true, account: false }); // 默认参数:开发服务器端口7296,启用热更新,不启用登录系统;
```
1. 恢复默认文件,也新增是否启用登录:
```js
initProject({ mode: 'skip-dirs', verbose: false, account: false}); // 默认参数:跳过已存在的目录,不启用详细日志,不启用登录系统;
```
1. 编译打包文件支持自定义打包目录名:
```js
compile({ outputDir: 'dist' }); // 默认参数:目录名 dist;
```
### 登录系统支持
- 支持用户名和邮箱登录;
- 支持密码,密码+2FA,密码+硬件验证

### 页面样式在线修改(需启用登录系统)
- 支持长按元素选择设置其各种属性样式(比如:边距,颜色,字体等等);
- 支持跳转到编辑器修改整个页面样式文件:
> - 编辑器功能: 颜色选择 · 编辑 · 自动补全 · 代码折叠 · 预览 · 保存应用· 取消

### 自定义拖动元素(默认支持主题图标和返回顶部图标)
- 登录系统额外支持编辑器和预览容器拖动

### 重大重构:
- 将包从 CJS 完全迁移至 ESM 模块系统，以适配未来发展趋势。
  **推荐使用 `import` / `export` 语法**，并确保项目 `package.json` 已设置 `"type": "module"`（或脚本使用 `.mjs` 后缀）;
  **如需保留 `require()` 调用**，请使用 Node.js ≥ 23.5.0（原生支持）,或 Node.js 22.12+ 并显式开启 `--experimental-require-module` 标志;
  （本文档所有示例已统一采用 ESM 标准）
